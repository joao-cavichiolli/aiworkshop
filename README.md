# AIWorkshop

Generative AI Training Platform built for TAIKAI builders.

> Submitted to **Hacklayer 4_1 MCP Edition** by LayerX

## What is it?

AIWorkshop is a learning platform integrated into TAIKAI that teaches builders and developers to master generative AI — from crafting efficient prompts to using market-leading MCPs — all within the context of building real hackathon projects.

## Features

- 📚 **Training Catalog** — 10 modules across 4 categories with filters
- ⚡ **Prompt Engineering** — Beginner to advanced prompt techniques
- 🔌 **MCP Tools** — Hands-on tutorials for real-world MCP servers
- 🤖 **AI Agents** — Agent patterns and multi-agent orchestration
- 🏆 **Hackathon Ready** — Practical AI strategies for time-constrained builds

## Tech Stack

- [Next.js 15](https://nextjs.org/) — App Router
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/)

## Getting Started

```bash
npm install
npm run dev
```
